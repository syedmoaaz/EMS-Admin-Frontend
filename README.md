# EMS Admin Frontend

Employee Monitoring System (EMS) - Admin Dashboard built with **React**, **Vite**, and **Tailwind CSS**.

This repository contains the web-based administration panel for monitoring employees, attendance, and field staff activities across multiple branches.

---

## 🚀 Project Overview

EMS is designed for a pharmaceutical distribution company to monitor:

- Office employee attendance
- Field order takers
- Dispatchers
- Live employee locations
- Attendance reports
- Branch-wise employee management

---

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Router DOM
- Axios
- Lucide React
- React Icons
